基础 ​

显示代码
vue
<lux-switch v-model="value"></lux-switch>

自定义开启/关闭值 ​

显示代码
vue
<lux-switch v-model="modelValue" active-value="on" inactive-value="off" @change="handleChange"></lux-switch>

文字描述 ​

否是
显示代码
vue
<lux-switch v-model="valueText" active-text="是" inactive-text="否"></lux-switch>

禁用 ​

显示代码
vue
<lux-switch v-model="modelValueDisabledTrue" disabled></lux-switch>
<lux-switch v-model="modelValueDisabledFalse" disabled></lux-switch>

加载 ​

显示代码
vue
<lux-switch v-model="modelValueLoading" :loading="loading" @change="handleSwitchLoading"></lux-switch>

自定义背景颜色 ​

显示代码
vue
<lux-switch v-model="modelValueColor" active-color="#F9F871" inactive-color="#E9EFFF"></lux-switch>
